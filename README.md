# SmartProperty
![](logo_min.png)

这是一个探索智能房产租赁的项目。

房屋所有人可以在区块链上提交一个房屋租赁需求，系统自动生成的智能合约，租户可以申请这个智能合约，提交日期和租金，便能获得租期内该房屋门锁的控制权。这个项目还可以从房屋租赁拓展到车位短期租赁，等各种共享经济类的应用场景。
